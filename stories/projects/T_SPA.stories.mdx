# Static Program Analyzer (SPA)
## CLI app that analyzes source code (Simple SOURCE language) and stores its metadata for querying

<br />

#### Tech Stack: C++17, Microsoft Visual Studio 2020, GitHub Actions

- Formulated a CLI app that analyses program source code, stores its metadata and allows for querying of keywords
- Developed a query processor that parses queries and selects relevant information from program knowledge base
- Setup continuous integration pipeline using GitHub Actions to automate a set of unit, integration and system tests
