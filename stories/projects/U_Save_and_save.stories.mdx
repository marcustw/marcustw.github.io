# Save & Save
## Web App listing leftovers from supermarkets in effort to reduce food wastage

<br />

#### Tech Stack: Python, React, PostgreSQL, Amazon Web Services

- Developed a web app using the PaaS model on AWS Amplify for supermarkets in effort to reduce food wastage
- Built an API Gateway REST API with AWS Lambda proxy integrations developed using Python and Amazon RDS
